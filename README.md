# HashRental — GitHub Pages Site (Production-ready)

This repo contains a production-ready landing page for HashRental (Nordhash Oy).

Features
- Mobile-first, accessible (WCAG AA), fast static site (no frameworks, no build tools).
- Apple/Tesla inspired dark glassmorphism design.
- SEO: canonical, Open Graph, Twitter Card, keywords, Organization & WebSite JSON-LD, FAQ schema, sitemap, robots.
- Conversion-focused: prominent hero, pricing, trust badges, How It Works, specs, social proof, expanded FAQ, clear CTAs.
- Performance: lazy images, small DOM, no external fonts/resources, minimal JS.

Deployment
1. Create a GitHub repo and push all files to the repo root.
2. Add a `favicon.ico` to the repo root (optional — add your preferred icon).
3. In GitHub → Settings → Pages, set source to the default branch and root (`/`).
4. (Optional) Add custom domain in Pages settings and update canonical/OGs if needed.

Notes
- Canonical and Open Graph URLs are set to `https://hashrental.store/` per request.
- All payment CTAs use SumUp links provided.
