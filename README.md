# ⚡ Hashrate Rent

A simple platform for renting real SHA-256 mining hashrate.

## Features

- Rent real ASIC hashrate
- 720 hour rental periods
- SHA-256 algorithm only
- Instant delivery
- Mobile friendly
- Live availability
- Profit calculator
